# D&D Кубики

Qt 6 приложение для бросков D&D 5e.

## Быстрый старт

| Файл | Действие |
|------|----------|
| **build.bat** или **СОБРАТЬ.bat** | Сборка (копия в `C:\DndDice`, без кириллицы в пути) |
| **ПРОВЕРКА.bat** | Статус проекта и Qt |
| **ЗАПУСК.bat** | Запуск игры (после сборки) |

Готовый exe: `deploy\DndDice.exe` или `C:\DndDice\deploy\DndDice.exe`

## Важно

Папка `C:\D&D Кубики` содержит `&` и кириллицу — **MinGW не собирает здесь напрямую**.  
Скрипт **build.bat** копирует проект в **C:\DndDice** и собирает там.

Папку **build_qt** в этом каталоге можно **удалить** (старая неудачная сборка).

## Qt

`C:\Qt\6.8.3\mingw_64` + MinGW 13.1 + CMake из Qt Tools.

## Функции

- d4–d20, количество кубов, преимущество/помеха d20
- Прокид характеристик (6×4d6, без низшего)
